# 🎯 Sobre o Projeto

O **ArcadeVerse Premium** é um portal arcade desenvolvido utilizando HTML5, CSS3 e JavaScript puro, contendo três jogos clássicos totalmente funcionais e interativos.

O principal objetivo do projeto foi aplicar conceitos fundamentais de desenvolvimento Front-End, lógica de programação, manipulação do DOM e criação de interfaces dinâmicas sem recarregar a página.

Os jogos disponíveis são:

- ✊ Pedra, Papel ou Tesoura (Jokenpô)
- 🎲 Batalha de Dados
- 🪙 Cara ou Coroa

---

# 💻 Tecnologias Utilizadas

- **HTML5:** Estruturação semântica das páginas.
- **CSS3:** Estilização visual, responsividade e animações.
- **JavaScript (ES6):** Manipulação do DOM, lógica dos jogos e eventos.

# ✊✋✌️ Jogo 1 — Pedra, Papel ou Tesoura (Jokenpô)

## 🎮 Sobre o Jogo

Este jogo representa a versão digital do clássico Pedra, Papel ou Tesoura, onde o jogador enfrenta uma Inteligência Artificial desenvolvida em JavaScript.

O sistema utiliza `Math.random()` para gerar as escolhas da IA, além de atualizar o placar e os resultados dinamicamente na tela.

### ✨ Funcionalidades

- Interface interativa com imagens das jogadas.
- Inteligência Artificial aleatória.
- Sistema de vitória, derrota e empate.
- Placar dinâmico.
- Animações durante as partidas.
- Atualização do DOM sem refresh.

---

## 🎯 Como Jogar

1. Escolha:
   - Pedra
   - Papel
   - Tesoura
2. A IA fará sua jogada automaticamente.
3. O resultado será exibido na tela.
4. O placar será atualizado em tempo real.

# 🎲 Jogo 2 — Batalha de Dados

## 🎮 Sobre o Jogo

Neste jogo, o jogador disputa contra a Inteligência Artificial através de lançamentos aleatórios de dados.

O sistema sorteia números de 1 a 6 utilizando JavaScript e determina automaticamente o vencedor da rodada.

### ✨ Funcionalidades

- Rolagem aleatória de dados.
- Animação de lançamento.
- Verificação automática de vencedor.
- Interface responsiva.
- Atualização dinâmica da tela.

---

## 🎯 Como Jogar

1. Clique em **Lançar Dados**.
2. O sistema realizará os sorteios.
3. O maior valor vence a rodada.
4. Se os valores forem iguais, ocorre empate.

# 🪙 Jogo 3 — Cara ou Coroa

## 🎮 Sobre o Jogo

O clássico Cara ou Coroa foi recriado digitalmente utilizando JavaScript e manipulação dinâmica de imagens.

O jogador escolhe um lado da moeda enquanto o sistema realiza um sorteio aleatório.

### ✨ Funcionalidades

- Escolha entre Cara ou Coroa.
- Sorteio utilizando `Math.random()`.
- Troca dinâmica de imagens.
- Animação da moeda.
- Resultado instantâneo na tela.

---

## 🎯 Como Jogar

1. Escolha:
   - Cara
   - Coroa
2. O sistema realizará o sorteio automaticamente.
3. O resultado será exibido imediatamente.
